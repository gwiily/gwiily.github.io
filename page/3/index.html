在模拟器中安装google play版部落冲突

Posted on 2016-05-15 Edited on 2018-07-07 In Play Valine:

想在模拟器中玩部落冲突，但是在google市场里无法安装部落冲突以及皇室战争，显示“您的设备不兼容”，只需要修改一个文件即可。步骤如下：

  1. 首先模拟器必须自带root或者可以root，安装re管理器
  2. 进入re管理器，./system目录，找到build.prop，长按选择以文本编辑的方式打开
  3. 找到ro.product.model=xxx,修改xxx为手机型号，如GT-i9100,等，以及其他参数也修改一下，如name
  4. 修改完成后右上角选择保存并退出，重启模拟器即可

Crazyflie2.0 文件结构

Posted on 2016-05-09 Edited on 2018-07-07 In Quadcopter , Crazyflie 2.0 Valine:

./ | 根目录，包含Makefile

+ init         | 主函数
+ config       | 配置文件 
+ drivers      | 硬件驱动文件
|  + src       | 驱动文件源代码 
|  + interface | 驱动文件头文件. Interface to the HAL layer
+ hal          | 硬件抽象层 
|  + src       | HAL 源代码
|  + interface | HAL 头文件及接口
+ modules      | 固件操作源码和头文件
|  + src       | 固件子函数
|  + interface | 头文件. 配置固件环境
+ utils        | 工具包代码. 实现诸如控制台的程序.
|  + src       | 工具包源码
|  + interface | 工具包头文件及接口
+ platform     | 固件的特殊文件，不一定被使用
+ tools        | 杂项. scripts for LD, OpenOCD, make, version control, ...
|              | *** 这两个以下文件夹中包含未修改文件 ***
+ lib          | 库文件
|  + FreeRTOS  | FreeRTOS 源码， 删除了不必要的问题
|  + STM32...  | STM32的外围设备的库文件
|  + CMSIS     | 核心抽象层

./ | Root, contains the Makefile

+ init         | Contains the main.c
+ config       | Configuration files
+ drivers      | Hardware driver layer
|  + src       | Drivers source code
|  + interface | Drivers header files. Interface to the HAL layer
+ hal          | Hardware abstaction layer
|  + src       | HAL source code
|  + interface | HAL header files. Interface with the other parts of the program
+ modules      | Firmware operating code and headers
|  + src       | Firmware tasks source code and main.c
|  + interface | Operating headers. Configure the firmware environement
+ utils        | Utils code. Implement utility block like the console.
|  + src       | Utils source code
|  + interface | Utils header files. Interface with the other parts of the program
+ platform     | Platform specific files. Not really used yet
+ tools        | Misc. scripts for LD, OpenOCD, make, version control, ...
|              | *** The two following folders contains the unmodified files ***
+ lib          | Libraries 
|  + FreeRTOS  | Source FreeRTOS folder. Cleaned up from the useless files 
|  + STM32...  | Library folders of the ST STM32 peripheral libs 
|  + CMSIS     | Core abstraction layer

crazyflie2.0介绍

Posted on 2015-11-29 Edited on 2018-07-07 In Quadcopter , Crazyflie 2.0 Valine:

Crazyflie2.0是一个多功能的飞速发展平台，它的重量只有27克，适合在你的手掌。其先进的功能，使得它非常适合开发人员和蓝牙低功耗（LE）功能，可以很容易地从移动设备飞行。由于其体积小，重量轻，这是非常适合室内使用，但也可以很容易上面盘旋你的房子，因为它可以在你的餐桌上徘徊。配件设计为无需焊接，只通过简单的连接到电机的电路板，就可以准备起飞了。
